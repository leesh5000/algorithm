### Problem 1

- 문제: 자바(Java) 알고리즘 문제풀이 입문: 코딩테스트 대비 (1. 문자찾기)
- 문제 유형: 문자열
- 풀이 방법: 
  - 문자열을 순회하면서 각 문자의 개수를 세는 방법으로 풀이
  - 대소문자를 구분하지 않기 위해 모두 소문자로 변환하여 비교
- 체크포인트:

### Problem 2
